# Markdown 分享示範

這份檔案示範 GitHub 原生渲染 Markdown 能附的東西：表格、code block、checklist、Mermaid 流程圖、引言、連結。

## 表格

| 項目 | 說明 | 狀態 |
|---|---|---|
| 表格 | 原生渲染，欄位對齊 | ✅ |
| Code block | 語法高亮 | ✅ |
| Mermaid | 流程圖/序列圖/甘特圖 | ✅ |
| 圖片 | 相對路徑或絕對 URL | ✅ |
| Checklist | 可勾選 (靜態) | ✅ |

## Code block

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("world"))
```

## Checklist

- [x] 支援表格
- [x] 支援語法高亮
- [x] 支援 Mermaid
- [ ] 不支援互動 (hover / 點擊 / 動畫) — 這點是 Markdown 的已知限制

## Mermaid 流程圖

```mermaid
flowchart LR
    A[本機檔案] --> B[commit]
    B --> C[CI 處理]
    C --> D[公開連結]
```

## 引言

> Markdown 渲染是平台原生的，不需要額外的 Pages 服務 — 這是它比 HTML 更快能用的原因。
